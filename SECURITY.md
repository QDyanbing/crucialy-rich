# Security Policy

当前项目处于早期脚手架阶段，暂未发布稳定版本。

## 支持范围

| 版本    | 状态   |
| ------- | ------ |
| `0.0.x` | 开发中 |

## 报告安全问题

请不要在公开 Issue 中粘贴敏感信息、凭据、访问令牌或私有日志。

当前仓库没有单独的安全邮箱配置时，请先通过私有渠道联系仓库维护者；如果只能使用 Issue，请只描述影响范围和复现条件，不要公开漏洞利用细节或敏感数据。

## 处理原则

- 优先确认影响范围和可复现性。
- 修复应包含验证步骤。
- 涉及依赖或 CI 的问题，应说明受影响的命令和环境。
